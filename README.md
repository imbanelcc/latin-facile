# 🏛️ Latin Facile

Site web interactif pour apprendre le latin facilement !

## ✨ Fonctionnalités

- 📚 **Tableaux de déclinaisons** : 5 déclinaisons complètes
- ✍️ **Tableaux de conjugaisons** : Présent et imparfait, voix active et passive
- 📖 **Lexique interactif** : Recherche de mots, verbes et adjectifs
- 🎯 **Exercices** : Génération automatique d'exercices pour s'entraîner

## 🚀 Installation

1. Cloner le repository
2. Ouvrir `index.html` dans un navigateur
3. C'est tout ! Aucune installation requise.

## 📱 Utilisation

Le site est entièrement responsive et fonctionne sur tous les appareils.
